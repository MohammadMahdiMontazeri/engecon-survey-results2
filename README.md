# Engineering Economics Final Project Results

This repository hosts a static website for presenting the results of the final project activities in the **Engineering Economics** course.

The website is designed to provide students with a simple and organized way to review the outcomes of the project survey process.

## About the Course

Engineering Economics focuses on evaluating engineering projects from an economic perspective. Students apply concepts such as cost estimation, revenue analysis, cash flow modeling, investment evaluation, and decision-making criteria to real-world project ideas.

As part of the final project, student groups developed and presented business or engineering-economic ideas and received feedback through a structured survey process.

## Website Overview

The website includes the following main sections:

### Individual Results

Students can check their individual survey participation status using their student ID.

### Group Results

This section presents a summary of the group evaluation results based on the survey criteria.

### Strengths and Weaknesses

This section displays the qualitative feedback submitted for each group, including strengths and areas for improvement.

## Deployment

The website is built as a static site and is published using GitHub Pages.

## Local Preview

To preview the website locally, run:

```bash
python -m http.server 8000 --directory docs
```

Then open:

```text
http://localhost:8000
```

## Repository Structure

The public website files are located in the `docs/` directory.
